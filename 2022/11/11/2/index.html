魂系关卡/地图编辑工具DSmapStudio试用

发表于2022-11-11|更新于2023-04-25
|阅读量:

下载地址

https://github.com/soulsmods/DSMapStudio

宫崎英高的游戏mod社区工具代代相传，从黑暗之魂到只狼，从恶魔之魂到埃尔登法环，
你都可以用DSmapstudio/DSAnimStudio来编辑游戏的地图/数值/动作/判定等等

食用指南

第一步：解包

在使用DSMapstudio之前，我们还需要用UXM解包工具来解包艾尔登法环
请注意，在解包完成后法环会变成原来两倍大小（大约100G）

下载链接：
https://www.nexusmods.com/eldenring/mods/1651
下载完后放置在任意位置，直接运行uxm.exe即可
在Executable path中选定游戏目录里的elden ring.exe(以艾尔登法环为例)

选中路径之后，点击unpack即可

第二步：理清你想合并的regulation.bin

你想合并regulation.bin，那你肯定已经在用mod了，所以我们以你正在使用的mod为基础
添加一个新的mod
新的mod叫mod2,你已经在使用的mod叫mod1

第三步：new 一个project

在DSMapstudio左上角的file里找到new project

Project name取一个你想要的名字，比如Main
Project directory 选取你的mod1的regulation.bin
Game Executable选取你的游戏启动方式，elden ring.exe
记住勾选Import row names
然后点击Create即可

然后你就创造了一个你的mod1的project
接下来我们按照同样的方法创建一个mod2的project
Project name可以取你的mod2的名字，比如Grand Merchant
